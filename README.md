# 🛡 ОхранаТруда.PRO

**B2B-платформа для управления документацией по охране труда**  
🚀 Lead Magnet система → Бесплатные шаблоны → Продажа подписки

Единый репозиторий для веб-клиента (Vite), API-сервера (Express) и Telegram-бота авторизации.

## 🔥 Ключевые особенности

- **🔐 Безпарольная авторизация:** Вход через телефон + подтверждение в Telegram боте.
- **🎁 Lead Magnet:** Бесплатный доступ к базовым шаблонам документов для привлечения клиентов.
- **💼 B2B Подписки:** Уровни доступа (Free, Starter, Business, Enterprise) с контролем сроков.
- **📊 Аналитика:** Логирование скачиваний, поиск и избранное для каждого пользователя.
- **⚡ Быстрый старт:** SQLite для данных, TypeScript для надежности.

## 🏗 Архитектура и Стек

| Компонент | Технологии | Назначение |
|---|---|---|
| **Frontend** | Vite, TypeScript | Витрина документов, ЛК клиента |
| **Backend** | Express, TypeScript | API, бизнес-логика, JWT |
| **Bot** | Telegram Bot API | Авторизация, уведомления менеджерам |
| **DB** | SQLite | Пользователи, сессии, документы, логи |

## 🔐 Флоу Авторизации (Passwordless)

1. **Ввод телефона:** Клиент оставляет номер на сайте.
2. **Генерация ссылки:** Сервер создает сессию и ссылку `t.me/bot?start=TOKEN`.
3. **Подтверждение:** Клиент запускает бота → Бот подтверждает личность через Telegram ID.
4. **JWT Токен:** Сервер выдает доступ, клиент залогинен ✅.
